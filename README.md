# LuModa

LuModa es un e-commerce desarrollado en ReactJS como parte del curso de React JS de CoderHouse.

## Funcionalidades

* Navegación mediante React Router.
* Catálogo principal de productos.
* Filtrado de productos por categorías.
* Vista de detalle de producto.
* Componente ItemCount para futuras funcionalidades de carrito.
* Página 404 para rutas inexistentes.
* Diseño responsive utilizando TailwindCSS y DaisyUI.

## Tecnologías utilizadas

* ReactJS
* Vite
* React Router DOM
* TailwindCSS
* DaisyUI

## Estructura del proyecto

* ItemListContainer: manejo del listado de productos.
* ItemDetailContainer: manejo del detalle de productos.
* ItemList, Item e ItemDetail: componentes de presentación.
* Navbar y CartWidget para la navegación.
* Products.js como fuente de datos simulada mediante Promises.
